# Scraper
Using beautifulsoup and selenium to scrape through Walmart and eBay,
and making a pandas data frame visualize the data of the products.

## Installation
with the package manager [pip](https://pip.pypa.io/en/stable/) to install 
- beautifulsupe
- reuests
- pandas
- selenium

```bash
pip install beautifulsupe, reuests, pandas, selenium 
```

## Usage
get the data faster and visulize it with pandas!

### Note 
you can adjust the numper of the showed row in (make_df) function.
